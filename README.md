# Barnstable MTSS Guidebook

This folder is self-contained for GitHub/Vercel deployment.

Required files:

- `index.html`
- `styles.css`
- `script.js`
- `assets/barnstable-logo.png`

## Vercel setup

If the repo contains only this website, deploy from the repo root.

If the repo contains other folders too, set Vercel's **Root Directory** to:

`mtss-guide`

Do not upload only `index.html`; the page needs `styles.css`, `script.js`, and the `assets` folder.
